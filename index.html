Animation Libary

Made in SASS, CSS and pain

Github

How to use:

Import the css binary: (https://css-animation-libary.sean-outram.dev/main.css)
then simply use one of the classes on the element you want to animate!

Classes's

blink

Class Description
blink-variation1 fast blink
blink-variation2 slower blink

bounce

Class Description
bounce-bottom bounces downwards
bounce-left bounces left
bounce-right bounces right
bounce-top bounces up

jello

Class Description
jello-horizontal has a jelly like effect horizontaly
jello-vertical has a jelly like effect vertical

pulse

Class Description
pulse-heartbeat pulses like a heartbeat
pulse-ping pulses like a ping
pulse-regular pulses

shake

Class Description
shake-bottom shakes downwards
shake-horizontal shakes horizontaly
shake-left shakes left
shake-right shakes right
shake-rotate shakes in a rotation
shake-top shakes upwards
shake-vertical shakes vertically

vibrate

Class Description
vibrate-variation1 a vibrate animation
vibrate-variation2 another vibrate animation
vibrate-variation3 another vibrate animation

wobble

Class Description
wobble-bottom wobbles downwards
wobble-left wobbles left
wobble-right wobbles right
wobble-top wobbles upwards
